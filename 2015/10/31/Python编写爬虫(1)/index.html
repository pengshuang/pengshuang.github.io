Python编写爬虫(1)

正则表达式的应用举例

  1. 使用findall与search从大量文本中匹配感兴趣的内容

  2. 使用sub实现换页功能

匹配多段内容

  1. 灵活使用findall与search

注意 ： findall是尽可能多的去找符合要求的，而search找到一个之后就不再找了

  1. 先抓大再抓小

核心代码

new_link = re.sub(‘pageNum=\d+’,’pageNum=%d’%i, old_url)

测试文件：

<html>
  <head>
    <title>小沙文爬虫测试</title>
  </head>
  <body>
    <div class="topic"><a href="http://opencourse.com/welcome.html">欢迎参加</a>
      <div class="list">
        <ul>
          <li><a href="http://bbs.byr.cn/1.html">这是第一条</a></li>
          <li><a href="http://bbs.byr.cn/2.html">这是第二条</a></li>
          <li><a href="http://bbs.byr.cn/3.html">这是第三条</a></li>
        </ul>
      </div>
    </div>
  </body>
</html>

测试代码:

#-*-coding:utf8-*-

#导入re库文件
import re

old_url = 'http://www.opencourse.com/course/?pageNum=2'
total_page = 20

f = open('text.html','r')
html = f.read()
f.close()

#爬取标题
title = re.search('<title>(.*?)</title>',html,re.S).group(1)
print title

#爬取链接
links = re.findall('href="(.*?)"',html,re.S)
for each in links:
    print each

#抓取部分文字,先大再小
text_fied = re.findall('<ul>(.*?)</ul>',html,re.S)[0]
the_text = re.findall('">(.*?)</a>',text_fied,re.S)
for every_text in the_text:
    print every_text

#sub实现翻页
for i in range(2,total_page+1):
    new_link = re.sub('pageNum=\d+','pageNum=%d'%i,old_url,re.S)
    print new_link
