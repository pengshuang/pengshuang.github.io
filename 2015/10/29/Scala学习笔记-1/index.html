Scala学习笔记(1)

最近因为做测试要用到Spark，所以在学Scala，下面记录一下学习过程中的笔记。

关于IDE的配置和Scala下就不在赘述了，相关的网上教程很多，写下来太花时间，主要写一些Scala的基础语法和高级特性。

值与变量

值(val):赋值后不可变
val 值名称:类型 = xxx

变量(var):赋值后可以改变
var 变量名称:类型 = xxx

一般不需要显式指定类型,因为可以从赋值中推断出类型

常用类型

Byte

Char

Short

Int

Long

Float

Double

Boolean

并⽆基本类型与包装类型之分

方法定义

def ⽅法名(参数名 : 参数类型) : 返回类型 = { 

//block内最后一行为返回值

}

当返回值为Unit时可以定义为: def ⽅方法名(参数名 : 参数类型) { }

方法注意

  1. 没有参数的⽅方法可以不带圆括号访问

  2. Scala没有静态⽅方法,通过object来实现

基础知识

  1. 条件表达式(if)

  2. 循环表达式(for,while,to,until,Range没有continue与break)

  3. 语句终⽌止(分⾏行写时可以不⽤用分号)

  4. ￼Lazy value

    用到的时候才会去初始化

  5. 默认参数

  6. 带名参数（赋值时顺序可以跟定义时的顺序不一样）

  7. 变长参数(:_*)

       def sum(args: Int*) = {
    var result = 0
   for( arg <- args) result += arg
    result
}

    星号表示可接受变长参数列表。函数实际得到的参数是Seq类型的。

注意，在调用的时候需要：

val s = sum(1, 4, 9, 16, 25) //true
val s = sum(1 to 5) // false
val s = sum(1 to 5: _*

如果sum函数被调用时传入的是单个参数，那么该参数必须是单个整数，而不是一个整数区间。解决这个问题的办法是告诉编译器你希望这个参数被当做参数序列处理。追加: _*，就像上面这样：

def recursiveSum(args: Int*): Int = {  
  if(args.length == 0) 0  
 else args.head + recursiveSum(args.tail: _*)  
}

在这里，序列的head是它的首个元素，而tail是所有其他元素的序列，这又是一个Seq，我们用: _*来将它转换成参数序列。

定长数组

  1. val array_name = new Array[T]（length）

  2. val array_name = Array(“”,””)

  3. 通过( )访问,⽽不是[ ]

变长数组

  1. import scala.collection.mutable.ArrayBuffer

  2. val buff = ArrayBuffer[T]（）

  3. +=/++=/insert/remove/toArray/sum/max/reverse

遍历数组

  1. for(i <- 0 until array_name.length)

  2. for(i <- array_name)

  3. 事实上会更多的使⽤用map、filter等等来操作

Map(可变与不可变)

  1. 不可变 val age = Map(“Jack” -> 20,”Lucy” ->”18” )

  2. 可变 val age = scala.collection.mutable.Map( … )

  3. 初始化 val m =scala.collection.mutable.Map[String,Int]（）

Map操作

  1. 取值 map(key),更好的⽅方式map.getOrElse(key,default)

  2. 更新 map(key)=value / += / -=

  3. 迭代 for((k,v) <- map){ },也可单独取key或者value for((k,_) <- map) / for((-,v) <- map) 也可使⽤用map.keySet 或者 map.values

元组

  1. ( )⾥包含⼀系列的值

  2. 通过._ 取值,下标从1开始,例如 t._1

  3. 迭代 for(elem <- t.productIterator)
