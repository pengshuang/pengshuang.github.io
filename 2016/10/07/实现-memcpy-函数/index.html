实现 memcpy 函数

发表于 2016-10-07   |   分类于 面试总结   |  

1. 接口设计

void mymemcpy(void* dst, const void* src, size_t num)

2. 边界条件

典型错误

void mymemcpy(void* dst, const void* src, size_t num){
    assert((dst != NULL)&&(src != NULL));

    const char * psrc = (const char *)src;
    char * pdst = (char *)dst;
    while(num-- > 0)
    {
        *pdst++ = *psrc++;
    }
}

情况1：不重合， 没问题

情况2：重合，且dst < src，也没问题

情况3：重合，且dst >= src，oops！

解决方法：从后向前拷贝

正确解法

void mymemcpy(void *dst, const void *src, size_t num)
{
    assert((dst != NULL)&&(src != NULL));

    const char* psrc = (const char*)src;
    char* pdst = (char*)dst;

    if (pdst > psrc && pdst < psrc + num)
    {
        for (size_t i = num - 1; i != -1; --i)
        {
            pdest[i] = psrc[i];
        }
    else
    {
        for (size_t i = 0; i < num; ++i)
        {
            pdst[i] = psrc[i];
        }
    }
}

3. 如何对memcpy函数进行优化

关键点：没有必要一个字节一个字节拷贝

优化:

void mymemcpy(void *dst, const void *src, size_t num)
{
    assert((dst != NULL)&&(src != NULL));

    int wordnum = num / 4;
    int slice = num % 4;
    const int* psrc = (const int*)src;
    int* pintdst = (int*)dst;

    // 一次拷贝 4 bytes
    while(wordnum--)
    {
        *pintdst++ = *pintsrc++;    
    }

    // 拷贝剩下的bytes
    const char* pcharsrc = (const char*)pintsrc;
    char* pchardst = (char*)pintdst;
    while (slice--)
    {
        *pchardst++ = *pcharsrc++;
    }
}
